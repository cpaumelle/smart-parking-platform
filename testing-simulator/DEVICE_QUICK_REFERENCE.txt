WORKFLOW FOR EACH DEVICE:
1. Applications → Select application → + Add Device
2. Fill: Name, Description, DevEUI, Device Profile
3. Check: Skip frame-counter check
4. Submit
5. Keys (OTAA) tab → Set Device Keys
6. Enter AppKey (or generate random)
7. Submit

TEST AFTER FIRST DEVICE:
cd /opt/smart-parking/testing-simulator
sudo venv/bin/python3 test_mqtt_uplink.py

CHECK RESULTS:
- ChirpStack UI: Device page → LoRaWAN frames tab
- Ingest logs: sudo docker compose logs -f ingest-service
- Transform API: curl http://localhost/v1/devices | jq '.[] | select(.deveui | startswith("TEST"))'
